# 一.git项目的组织
---------------------------------
01.mobile 移动前端,前端公共组件库
02.web    PC前端,前端公共组件库
03.mvc   后端  api,logic,dblogic,model
04.infra 基础设施 框架 utility,tools
05.db    存放数据库的信息,sql脚本
06.doc   开发过程中的文档
